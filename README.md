# Trabajo Integrador Programación 1 
Sistema de gestión de datos de países desarrollado en Python. Permite cargar información desde un archivo CSV y realizar búsquedas, filtros, ordenamientos y estadísticas a través de un menú interactivo en consola.

## Descripción
El programa gestiona un dataset de países con los siguientes datos: nombre, población, superficie y continente. Los datos se leen y guardan en un archivo CSV llamado paises.csv.

## Opciones disponibles

- Agregar un país
- Actualizar población y superficie
- Buscar por nombre
- Filtrar por continente, rango de población o superficie
- Ordenar por nombre, población o superficie
- Ver estadísticas generales
- Listar todos los países
